Passo a passo para instalar o Polly MC e o Radmin para jogar Minecraft pirata com os amigos

Instalação PollyMC

IMPORTANTE NÃO CONFUNDIR PolyMC com PollyMC

Entre em: https://github.com/fn2006/PollyMC/releases
Procure por "PollyMC-Windows-MSVC-Setup-8.0.exe"
Baixe, execute e instale

Vá no canto superior esquerdo em "Adicionar instância"
Selecione a ultima versão (1.21.8)
* aqui tambem é onde da pra instalar mods e modpacks

Caso de o erro: "The java binary" couldn't be found. Please set up java in the settings."
Entrar em: https://www.java.com/pt-br/download/manual.jsp
Procure por "Windows On-line"
Baixe, execute e instale
No PollyMC vá em Configurações - Java - Runtime do Java e em "Caminho do Java"
Preencha com "C:\Program Files (x86)\Java\jre1.8.0_461\bin\java.exe" e teste

Esse caminho para o "java.exe" pode mudar, mas o padrão é esse. Em especial essa parte marcada muda de acordo com a versão do java que seja instalda.

Instalação Radmin VPN (para jogar online)
Entre em: https://www.radmin-vpn.com/
Clique em "Free Download"
Baixe, execute e instale
