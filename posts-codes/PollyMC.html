Esse é um passo a passo para instalar o PollyMC e o Radmin para jogar Minecraft pirata com os amigos

Instalação PollyMC

IMPORTANTE NÃO CONFUNDIR PolyMC com PollyMC
  1. Entre em: https://github.com/fn2006/PollyMC/releases
  2. Procure por "PollyMC-Windows-MSVC-Setup-8.0.exe"
  3. Baixe, execute e instale
  1. Vá no canto superior esquerdo em "Adicionar instância"
  2. Selecione a ultima versão (1.21.8)
    1. aqui tambem é onde da pra instalar mods e modpacks

Caso de o erro: "The java binary" couldn't be found. Please set up java in the settings."

  1. Entrar em: https://www.java.com/pt-br/download/manual.jsp
  2. Procure por "Windows On-line"
  3. Baixe, execute e instale
  4. No PollyMC vá em Configurações - Java - Runtime do Java e em "Caminho do Java"
  5. Preencha com "C:\Program Files (x86)\Java\jre1.8.0_461\bin\java.exe" e teste
    1. Esse caminho para o "java.exe" pode mudar, mas o padrão é esse.
    2. Em especial essa parte marcada muda de acordo com a versão do java que seja instalda.
Instalação Radmin VPN (para jogar online)
  1. Entre em: https://www.radmin-vpn.com/
  2. Clique em "Free Download"
  3. Baixe, execute e instale

Aqui divide em duas partes que é o Host e o Client

  1. Host:
    1. Rede - Criar rede
    2. Crie a rede com um nome e uma senha
  1. Client:
    1. Rede - Entrar em rede
    2. Entre com o nome e a senha
